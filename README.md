## 📂 Structure
- `weekXX/python/` – Python exercises
- `weekXX/r/` – R exercises
- `capstone/` – final project

## 🛠️ Requirements
- Python 3.10+ (pandas, numpy, matplotlib, seaborn, scikit‑learn)
- R 4.2+ (tidyverse, ggplot2, dplyr, tidymodels)

## 🧪 How to run
- Python: `python week01/python/hello.py` or use Jupyter notebooks.
- R: `Rscript week01/r/hello.R` or open in RStudio.
